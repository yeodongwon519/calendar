# 달력 (Calendar)

빠른 입력에 초점을 맞춘 미니멀 월간 달력. 기존 캘린더처럼 여러 번 클릭할 필요 없이, **날짜 칸을 클릭하면 바로 입력**하고 Enter로 연속 추가할 수 있다.

🔗 **배포 주소**: https://yeodongwon519.github.io/calendar/

## 기능

- **빠른 입력**: 날짜 칸 클릭 → 바로 타이핑 → `Enter` 저장하고 계속 입력 → `Esc` 닫기
- **체크박스**: 항목의 네모 박스 클릭 = 완료 토글
- **인라인 수정**: 글자 클릭 → 그 자리에서 수정 (`Enter` 저장 / `Esc` 취소)
- **삭제**: 항목에 마우스 올리면 나타나는 `×` 버튼
- **칸 높이 고정**: 할일이 많아도 칸이 커지지 않고 `+N 더` 표시 → 클릭하면 그 날 전체 팝오버
- **다크/라이트 테마** 전환 (선택 기억)
- **사이드바 연간 달력**: 12개월 한눈에, 클릭으로 이동
- 데이터는 브라우저 `localStorage`에 저장

## 단축키

| 키 | 동작 |
|----|------|
| `←` `→` | 이전/다음 달 |
| `T` | 오늘로 |
| `Enter` | 입력 저장 후 계속 |
| `Esc` | 입력 닫기 / 수정 취소 |

## 실행

정적 사이트라 별도 빌드가 필요 없다.

```bash
# 로컬에서 열기
python -m http.server 8080
# → http://localhost:8080
```

또는 `index.html`을 브라우저로 바로 열어도 된다.

## 기술 스택

- 순수 HTML / CSS / JavaScript (의존성 없음)
- 저장: 브라우저 localStorage

## 아이폰

- **홈화면 앱(PWA)**: Safari에서 사이트 접속 → 공유 → "홈 화면에 추가". 주소창 없는 전체화면 앱으로 열리고, 폰에선 월간 30일 뷰 + 날짜 탭 팝오버로 입력·수정.
- **홈화면 위젯**: 오늘 일정을 작은 위젯으로 표시(읽기 전용). 설치법은 [`widget/WIDGET_SETUP.md`](widget/WIDGET_SETUP.md).
- **멀티기기 동기화**: 위젯용 클라우드(Cloudflare Worker+KV)를 연결하면 PC·아이폰 일정이 자동 동기화됨.
